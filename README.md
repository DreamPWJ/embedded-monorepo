## STM32嵌入式单片机程序 Embedded Development

### 开发工具: CLion

### 开发语言: C 、C++

### MCU单片机类型: STM32或ESP32

### 嵌入式硬件平台: Arduino

### 云平台: 阿里云IoT物联网平台

### MQTT服务器: EMQX
